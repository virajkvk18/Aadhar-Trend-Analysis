# Aadhar-Trend-Analysis
Aadhaar Enrolment &amp; Update Trends Analysis Unlocking critical insights from UIDAI Aadhaar enrolment and update datasets to detect regional failure hotspots, track biometric &amp; demographic data drift over time, and support improved identity authentication through data-driven anomaly detection and visualization.
# Aadhaar Trend Analysis

This project analyzes Aadhaar enrollment trends using public UIDAI data.

## Objective
To study year-wise and state-wise Aadhaar trends using data analysis.

## Tools Used
- Python
- Pandas
- Matplotlib

## How to Run
Run the Python scripts or Jupyter notebooks after installing dependencies.
